About Project

Brain Tumor Detection Using MRI Images

This project classifies brain MRI images using a trained Python deep learning model and displays prediction results with confidence score.

Project Objective

The main objective is to classify brain MRI images into tumor categories and provide fast prediction results in a simple medical-style interface.

Model Classes

The model predicts four categories: Glioma, Meningioma, Pituitary, and No Tumor.

Technology Stack

Angular is used for frontend, Node.js and Express for backend, and Python Flask is used to serve the trained model API.

System Flow

The user uploads an MRI image. The backend validates the image and sends it to the Python model API. The prediction is then shown on the frontend.
